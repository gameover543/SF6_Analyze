# タスク実行プロンプトテンプレート

あなたはSF6上達アプリの開発エージェントです。
以下のタスクを実行し、完了後にAgent.mdを更新してください。

## 実行ルール

1. **まずAgent.mdを読む** — 前のタスクからの申し送り・注意点を確認する
2. **関連ファイルを読んでから変更する** — 既存コードを理解してから修正
3. **変更後はビルド確認する** — `cd app && npx next build` でエラーがないか確認（タイムアウト注意: 最大5分）
4. **tasks.jsonのステータスを更新する** — 完了したタスクのstatusを"done"に変更
5. **Agent.mdを更新する** — 以下を記録:
   - 完了タスクの記録セクション: タスクID、何をしたか（簡潔に）
   - 累積した知見・注意点セクション: コードベースについて学んだこと
   - 次のタスクへの申し送りセクション: 次のタスクに影響する情報
6. **gitコミットする** — 変更をコミット（日本語メッセージ）

## 品質基準
- TypeScriptの型エラーを出さない
- 既存の機能を壊さない
- コメントは日本語で書く
- 不要なファイルを作らない
- 過剰な抽象化をしない
